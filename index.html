Introduction to Xnee

Xnee can record, distribute and replay X (X11) protocol data. This is useful for automated tests of applications or benchmarking of applications. ...think of it as a robot.

Xnee requires X11 and the RECORD- and XTest extensions

For more information, read the Xnee documentation, visit the GNU Xnee software page which will link you to Xnee project page, and the the homepage of Xnee

The mailing list info-xnee@gnu.org is for discussion of Xnee. Write to info-xnee@gnu.org if you want to join. Volume is typically low.

Online Manual is soon available at www.gnu.org/ Xnee can be found on http://ftp.gnu.org/gnu/xnee/ or one of the mirrors.
