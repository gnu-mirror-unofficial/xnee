XNEE(1)                                                   XNEE(1)

NAME
       xnee record, replays or distributes X11 data

SYNOPSIS
       xnee [_______]

DESCRIPTION
       The  program Xnee can record and replay an X session. Xnee
       plays.   Xnee  gets  copies  of X protocol data from the X
       server. These are either printed to file ( ______ mode) or
       replayed  and  synchronised ( ______ mode).  During record
       and replay Xnee can distribute the record/replayed  events
       to multiple displays.

OPTIONS
       --help, -h
              Print this message

       --display,d  displayname
              X server to contact (default is localhost)

       --flags, -flags
              Prints all flags/options xnee accepts

       --file, -f 
              Read data from file file_name (default is stdin)

       --out, -o 
              Redirect  all  Xnee  data  to file_name (default is
              stdout)

       --err, -e 
              Redirect all Xnee verbose output file_name (default
              is stderr)

       --plugin ,-p 
              Use plugin file file_name

       --all_events, -ae
              Intercept all events

       --no_expose, -ne
              Do not intercept (print) Expose NoExpose

       --first_last, -fl
              Print  only  first  and last of multiple successive
              MotionEvent

       --everything
              puter may crash !

       --loops , -l
              Intercept n numbers of data ( n<0 means forever)

       --k_log
              Intercept 1000     (one thousand) data

       --10k_log
              Intercept 10 000   (ten thousand) data

       --100k_log
              Intercept 100 000  (one hundred thousand) data

       --m_log
              Intercept 1000 000 (one million) data

       --verbose, -v
              Verbose printout

       --time, -t  
              Delay start of application for  seconds. Used
              to prevent recording of  KeyRelease  when  starting
              Xnee from terminal

       --binary, -b
              Obsolete option, don't use this

       --text, -t
              Store data output in text format (default)

       --version, -V
              Print product information

       --all_clients, -ac
              Record all client's data (default)

       --future_clients, -fc
              Record future client's data

       --human_printouts, -hp
              Prints human readable

       --record, -rec
              Set recording mode (default)

       --replay, -rep
              Set replaying mode

       --stop_key mod key, -sk
              When pressing modifier mod and key key Xnee exits

       --print_settings, -ps
              Prints Xnee settings and waits (for )

       --manpage
              erating man page

       --distribute, -di 
              Distribute recorded  or  replayed  events  to  LIST
              where LIST is comma separated list of displays

       --device_event_range, -devera  
              Set device event range to X_LIST

       --delivered_event_range, -devra    
              Set delivered event range to X_LIST

       --error_range, -erra    
              Set error range to X_LIST

       --request_range, -reqra 
              Set request range to X_LIST

       --reply_range, -repra   
              Set reply range to X_LIST

       --extension_request_major_range, -erqmar  
              Set extension request major range to X_LIST

       --extension_request_mainor_range, -erqmir  
              Set extension request minor range to X_LIST

       --extension_reply_major_range, -erpmar  
              Set extension reply major range to X_LIST

       --extension_reply_mainor_range, -erpmir  
              Set extension reply minor range to X_LIST

COMMENTS
       file_name
              Name of a file (e.g /tmp/user1_session.xnee)

       display_name
              Name of a display (e.g 192.168.1.12:0

       secs   Seconds (e.g 10)

       n      Integer number (e.g 100)

       LIST   Comma    separated    list    of    display    (e.g
              192.168.1.12:0,129.12.23.0:0)

       X_LIST Comma   separated    list    of    integers    (e.g
              12-23,35,45-56)

EXAMPLES
       xnee  -k_log -devera 2-6 -o /tmp/xnee.rec -e /tmp/xnee.log
       -v

              Writes  1000  data  to file /tmp/xnee.rec and dumps
              the verbose printout to /tmp/xnee.log

       xnee -rep -f /tmp/xnee.rec -v -e /tmp/xnee.log --no_sync

              Read data from /tmp/xnee.rec, replay it and verbose
              print to file /tmp/xnee.log For more examples, look
              at the EXAMPLES file

       xnee can record and replay an X session.xnee also has  the
       ability to

                distribute events to multiple displays.
