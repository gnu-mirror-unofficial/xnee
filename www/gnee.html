Gnee

Gnee is the GUI frontend to libxnee.

Description

With Gnee you can record and replay user actions jst as you can with xnee. You can also use Gnee to make Xnee project files that both xnee and Gnee can use. This makes it easier for people not so familiar with the X11 protocol to use Xnee (both xnee and Gnee). For more information about the project file, read the user manual.
Gnee requirements:
libxnee.
gtk
