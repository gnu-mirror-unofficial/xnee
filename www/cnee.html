cnee / description

Summary

cnee is a command line program using libxnee.
With cnee you can record and replay user actions just as you can with gnee.

Background

The whole projext started as being one command line program, cnee. cnee was initially called xnee. When the Xnee project got bigger and more used the idea of a GUI program came up (today gnee) we decided to split Xnee into two parts. One library (today libxnee) and one program (today cnee).

Features

cnee comes with all features that is offered by libxnee. The same features are offered by the GUI program gnee. There is, however, one major thing that you only can do with cnee:
do unattended and autamoatic regression test (e.g using cron)
This might seems to be a rather useless feature. It is not!
Main features:
- record an Xnee session file
- replay an Xnee session file
- create/write Xnee project files
- read Xnee project files
- distribute device events to multiple displays

Requirements

libxnee.
